脚本系统

用 JavaScript 实现你的自定义功能

基于 GraalVM GraalJS，支持 ECMAScript 2024，所有核心类预绑定，开箱即用。

📁 ~/.zen/liquidres/scripts/ 🔄 热重载 📦 35+ 事件钩子 🧩 设置项支持
示例脚本： 按 F6 在聊天栏发送问候 
var mod = createModule("Greeter", Category.MISC, {
  onKey: function(e) {
    if (e.isPressed() && e.getKeyCode() == 295)
      printChat("§aHello from script!");
  }
});
mod.setKey(295); // F6
