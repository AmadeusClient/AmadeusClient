~/ scripts — API 参考

基于 GraalJS 的 JavaScript 脚本引擎 · 隔离上下文 · 模块化注册 · 35+ 事件

概述 #

基本概念

📁 存放目录
~/.zen/liquidres/scripts/
所有 .js 文件自动扫描加载。
🧩 隔离上下文
每个脚本独立 GraalJS Context，变量/函数互不冲突。
🔌 模块化注册
createModule() 自动加入 ModuleManager，可在 ClickGUI 中开关、绑键。
🔄 热重载
编辑后 .script reload 即可生效，自动恢复模块状态。
⚡ 事件驱动
支持 35+ 种事件（Tick、Motion、Packet、Render…）
🌐 免 import
所有常用 Java 类型、事件类、工具类已预绑定。

控制台命令

命令说明
.script load <文件名.js>加载脚本（文件必须在 scripts/ 目录）
.script reload <脚本名>热重载指定脚本（不需要 .js 后缀）
.script reloadall热重载所有已加载脚本
.script unload <脚本名>卸载脚本（自动禁用并移除模块）
.script list列出所有已加载脚本
提示
脚本名 = 文件名去掉 .js。例如 CenterText.js 的脚本名为 CenterText。

createModule() — 创建脚本模块 #

语法

var mod = createModule("模块名", Category.MISC, {
  onTick: function(e) { /* ... */ },
  onMotion: function(e) { /* ... */ },
  onRender2D: function(e) { /* ... */ },
  onEnable: function() { /* ... */ },
  onDisable: function() { /* ... */ }
});
参数类型说明
namestring模块名称，显示在 ClickGUI 和 HUD 中
catCategory分类：COMBAT / MOVEMENT / PLAYER / RENDER / WORLD / MISC
hooksobject事件回调映射，key 为钩子名，value 为处理函数

返回值是一个 ScriptModule 实例，可直接调用 mod.setKey()、mod.setEnabled() 等方法。

高级用法

绑定快捷键

mod.setKey(295); // GLFW_F6

键码为 GLFW 键码。常用：F6 = 295、RSHIFT = 345、LCONTROL = 341。

创建设置项

var speed = new NumberSetting("Speed", 1.0, 0.1, 10.0, 0.1);
var mode = new ModeSetting("Mode", "Vanilla", ["Vanilla", "Hypixel"]);
mod.registerSetting(speed);
mod.registerSetting(mode);

// 在事件中读取设置
var s = speed.getValue(); // double
var m = mode.getSelected(); // string
设置项注册后自动显示在 ClickGUI
调用 mod.registerSetting(setting) 后，设置会自动出现在 ClickGUI 的模块设置面板中。

完整示例 #

CenterText.js — 屏幕中央渲染文字

var module = createModule("CenterText", Category.RENDER, {
  onRender2D: function(e) {
    var gui = e.guiGraphics();
    var sw = mc.getWindow().getGuiScaledWidth();
    var sh = mc.getWindow().getGuiScaledHeight();
    var title = "§bAmadeus §7× §aScripts";
    gui.drawString(mc.font, title,
      ((sw - mc.font.width(title)) / 2) | 0,
      ((sh / 2 - 20) | 0),
      -1, true);
  }
});
module.setKey(295); // GLFW_F6

ChatLog.js — 聊天消息日志记录

var mod = createModule("ChatLog", Category.MISC, {
  onChatReceive: function(e) {
    java.lang.System.out.println("[CHAT] " + e.getMessage().getString());
  },
  onChat: function(e) {
    if (e.getMessage().contains("badword")) {
      e.setCancelled(true);
      printChat("§c消息被拦截！");
    }
  }
});

注意事项 #

类型陷阱 — 颜色值超出 int 范围

JavaScript 中 0x90000000 的值是 2415919104，超出 Java int 最大值。用 | 0 强制转成 32 位有符号整数：0x90000000 | 0。

方法重载 — Double 不能转 float

JS 的 number 类型都是 Double，涉及 drawRoundedRect 等重载方法时，用 | 0 确保整数传入。

热重载提示

.script reload 使用 脚本名（不带 .js），如 .script reload CenterText。重载会自动恢复该脚本所有模块的启用/禁用状态。

键码说明

使用 GLFW 键码。常用：F6 = 295，RSHIFT = 345，LCONTROL = 341，GRAVE = 96。

预绑定覆盖了所有 client.amadeus.* 类

源码中 client.amadeus.* 下的所有工具类、事件类、管理器都已预绑定。构建后类名会被混淆，但预绑定的 Java.type 字符串会被 ASM 自动重写，脚本完全不需要关心混淆后的类名。
